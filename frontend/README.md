# Blogly Frontend

This project contains the frontend code for the Blogly application.

## Getting Started

1. Install dependencies:
   ```bash
   bun install
   ```
2. Start the development server:
   ```bash
   bun start
   ```

## Features

- Responsive design
- Blog post creation and editing
- User authentication

## Folder Structure

- `src/` - Main source code
- `public/` - Static assets
